# ai-osteoarthritis-DB

## 소개

이 프로젝트는 PDF 형식의 논문을 업로드하면 해당 논문이 골관절염(Osteoarthritis)과 관련된 논문인지 확인하고, 사용자가 원하는 정보를 프롬프트 기반으로 자동 추출하는 코드입니다.

논문을 하나씩 직접 읽고 필요한 정보를 찾는 과정을 줄이기 위해 만들었으며, 골관절염 관련 연구 논문을 선별하고 핵심 정보를 정리하는 작업을 자동화하는 것을 목표로 합니다.

## 주요 기능

- PDF 논문 업로드 및 텍스트 분석
- 골관절염 관련 논문 여부 판별
- 사용자가 입력한 프롬프트에 따라 필요한 정보 추출
- 논문별 주요 정보 정리
- 반복적인 논문 검토 작업 자동화

## 사용 방법

1. 분석할 PDF 논문 파일을 준비합니다.
2. 코드 실행 환경을 준비합니다.
3. PDF 파일을 업로드합니다.
4. 골관절염 관련 논문인지 확인합니다.
5. 추출하고 싶은 정보를 프롬프트로 입력합니다.

예시 프롬프트:
```text
이 논문에서 사용된 세포 유형, 지지체 정보(제조방법, 성분, 성분배합비), bioactiviy, cytotoxicity, biodegradability 정보
```
